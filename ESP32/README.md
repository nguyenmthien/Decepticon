# ESP32 codes
Control ESP32 via Wi-Fi with TCP/IP socket.

## REQUIREMENTS:
* ESP32 (of course)
* Two motors with encoder
* L298N as motor driver

## PINOUT
To motor driver:
* leftMotorIN1: 15
* leftMotorIN2: 2
* rightMotorIN1: 16
* rightMotorIN2: 17
* leftMotorPWM: 4
* rightMotorPWM: 5 
To encoders:
* leftEncoder: 2
* rightEncoder: 3
